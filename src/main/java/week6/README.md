## 상속(Inheritance)

* 상위 클래스의 모든 멤버를 하위 클래스가 물려 받는 것
* extends를 이용한다.
* 상속을 하지 않는 경우 자동으로 Object라는 클래스를 상속받게 된다.
* 코드의 간결성, 재사용성을 위해 사용한다.



```
public class 클래스이름 extends 상위클래스이름{
	//클래스 내용

}
```
